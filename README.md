# 🌤️ Weather Dashboard

A modern, beautiful and responsive Weather Dashboard built with HTML, CSS & JavaScript.

![Weather Dashboard Preview](https://via.placeholder.com/800x400?text=Weather+Dashboard+Preview)

## ✨ Features

- 🌍 Real-time weather for any city
- 📍 Auto-detect current location
- ⏰ Hourly forecast (24 hours)
- 📅 7-day weather forecast
- 🌡️ Temperature, humidity, wind speed, UV index, feels like
- 🌈 Beautiful weather icons & background
- 🌓 Dark / Light mode
- 📱 Fully responsive design

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox + Grid)
- Vanilla JavaScript (ES6+)
- OpenWeatherMap API

## 🧩 How to Use

1. Clone the repository
2. Get your free API key from [OpenWeatherMap](https://openweathermap.org/api)
3. Put your API key in `js/config.js`
4. Open `index.html` in browser
